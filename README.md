## 👨‍💻 About

I'm an **AI Engineer** focused on **agentic systems, retrieval, and LLM inference**.
At **Spice** ([thespice.ai](https://thespice.ai)) I built the company's AI products and
agents from the ground up — including **Pulse / Lucid**, a creator "content chief-of-staff"
agent harness ([try-pulse.ai](https://try-pulse.ai)), and **AskSpice**, conversational
search over **10,000+ podcast episodes** and **1M+ searchable chunks**.

My strongest work spans **agent-harness engineering, LLM orchestration, tool execution,
Retrieval-Augmented Generation, hybrid search, durable memory, automated decision loops,
and LLM inference engineering**.
